Place offline Swagger UI assets here:
- swagger-ui.css
- swagger-ui-bundle.js

You can download them from:
- https://cdn.jsdelivr.net/npm/swagger-ui-dist@5/swagger-ui.css
- https://cdn.jsdelivr.net/npm/swagger-ui-dist@5/swagger-ui-bundle.js

If the CDN is blocked, manually copy these files from another machine and put them in this folder.

After assets are present, open http://127.0.0.1:8000/docs to view the API docs.
